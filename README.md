## What We Build

OpenConstruct gives agents senses (vision, sonar, touch), lets them join fleets that self-organize through spectral conservation, and collaborate through PLATO knowledge rooms — all orchestrated by text. From ESP32 microcontrollers to DGX Spark clusters, every node speaks the same constraint-native language. Apache 2.0 licensed, 12 languages, one mission: agents that know themselves through each other.
