## Design Notes for FM

### Common Elements
- Dark background (`#1a1a2e` or deeper)
- Accent color per domain (not all the same)
- JetBrains Mono for code/monospace
- Inter for body text
- No "AI-powered" badges
- No generic three-column feature grids
- One clear CTA per page, max two

### Per-Domain Accents
| Domain | Accent | Why |
|--------|--------|-----|
| cocapn.ai | Cyan `#3498db` | Deep ocean bioluminescence |
| dmlog.ai | Amber `#f39c12` | Tavern warmth |
| fishinglog.ai | Seafoam `#1abc9c` | Water |
| playerlog.ai | Magenta `#e91e63` | Arcade neon |
| luciddreamer.ai | Lavender `#9b59b6` | Dream state |
| makerlog.ai | Rust `#e67e22` | Workshop tools |

### Responsive
- Mobile first
- Table of contents on long pages
- Collapsible sections
- No horizontal scroll ever
